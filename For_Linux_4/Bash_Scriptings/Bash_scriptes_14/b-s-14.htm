#эр эрышїшх т эхь ёёћыюъ эр эхёѓљхёђтѓўљшх єрщыћ, ш тћтюфшђ шѕ шьхэр.

linkchk () {
    for element in $1/*; do
    [ -h "$element" -a ! -e "$element" ] && echo \"$element\"
    [ -d "$element" ] && linkchk $element
    # брью ёюсющ, '-h' я№ютх№џхђ ёшьтюышїхёъшх ёёћыъш, '-d' -- ърђрыюуш.

#ш шэєю№ьрішџ ю яю№џфъх яюыќчютрэшџ ёъ№шяђюь.
################
for directory in $directorys; do
    if [ -d $directory ]
        then linkchk $directory

            echo "Яю№џфюъ шёяюыќчютрэшџ: $0 dir1 dir2 ..."
    fi
done
 
exit 0
