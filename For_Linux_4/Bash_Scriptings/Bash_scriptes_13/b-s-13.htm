Хёыш тћ шёяюыќчѓхђх Ћёђрэфр№ђэѓўЛ юсюыюїъѓ *NIX-ёшёђхьћ, тючьюцэю, тћ эх чэръюьћ ё ђръющ яюыхчэющ юёюсхээюёђќў bash ъръ ьрёёштћ. еюђџ ьрёёштћ т bash эх ђръ ъ№ѓђћ, ъръ т P-џчћърѕ (Perl, Python ш PHP) ш ф№ѓушѕ џчћърѕ я№юу№рььш№ютрэшџ, юэш їрёђю сћтрўђ яюыхчэћ.
Bash-ьрёёштћ шьхўђ ђюыќъю їшёыхээћх шэфхъёћ, эю юэш эх юсџчрђхыќэћ ъ шёяюыќчютрэшў, ђю хёђќ тћ эх фюыцэћ юя№хфхыџђќ чэрїхэшџ тёхѕ шэфхъёют т џтэюь тшфх. Ьрёёшт іхышъюь ьюцхђ сћђќ юя№хфхыхэ яѓђхь чръыўїхэшџ чряшёхщ т ъ№ѓуыћх ёъюсъш:


  arr=(Hello World)


Юђфхыќэћх чряшёш ьюуѓђ сћђќ юя№хфхыхэћ ё яюьюљќў чэръюьюую тёхь ёшэђръёшёр (юђ Схщёшър (фр я№юёђшђ ьхэџ Фхщъёђ№р  я№шь. ях№хтюфїшър) фю дю№ђ№рэр):


  arr[0]=Hello

  arr[1]=World



Я№ртфр, юс№рђэюх тћуыџфшђ ё№ртэшђхыќэю сюыхх ѓ№юфыштю. Хёыш эѓцэю юс№рђшђќёџ ъ юя№хфхыхээющ чряшёш, ђюуфр:


  echo ${arr[0]} ${arr[1]}


Шч ёђ№рэшіћ man:

"дшуѓ№эћх ёъюсъш эѓцэћ фыџ я№хфюђт№рљхэшџ ъюэєышъђют я№ш №рчтю№рїштрэшш яюыэћѕ яѓђхщ ъ єрщырь."

Ъ№юьх ђюую, фюёђѓяэћ ёыхфѓўљшх ёђ№рээютрђћх ъюэёђ№ѓъішш:

  ${#arr[0]}# Фышэр ях№тющ чряшёш (эѓьх№рішџ ё эѓыџ)


${!arr[*]}  ё№ртэшђхыќэю эютюх фюяюыэхэшх т bash ш эх џтыџхђёџ їрёђќў ю№шушэрыќэющ №хрышчрішш. быхфѓўљрџ ъюэёђ№ѓъішџ фхьюэёђ№ш№ѓхђ я№шьх№ я№юёђюую шёяюыќчютрэшџ ьрёёштр. Юс№рђшђх тэшьрэшх эр "[index]=value", §ђю яючтюыџхђ эрчэрїшђќ ъюэъ№хђэюх чэрїхэшх ъюэъ№хђэюьѓ эюьх№ѓ чряшёш.


#!/bin/bash

 

array=(one two three four [5]=five)

 

echo "Array size: ${#array[*]}"  # Тћтюфшь №рчьх№ ьрёёштр

 

echo "Array items:" # Тћтюфшь чряшёш ьрёёштр

for item in ${array[*]}

do

    printf "   %s\n" $item

done

 

echo "Array indexes:" # Тћтюфшь шэфхъёћ ьрёёштр

for index in ${!array[*]}

do

    printf "   %d\n" $index

done

 

echo "Array items and indexes:" # Тћтюфшь чряшёш ьрёёштр ё шѕ шэфхъёрьш

for index in ${!array[*]}

do

    printf "%4d: %s\n" $index ${array[$index]}

done

Юс№рђшђх тэшьрэшх, їђю ёшьтюы "@" ьюцхђ сћђќ шёяюыќчютрэ тьхёђю "*" т ъюэёђ№ѓъішџѕ ђшяр {arr[*]}, №хчѓыќђрђ сѓфхђ юфшэръютћь чр шёъыўїхэшхь №рчтю№рїштрэшџ чряшёш т ъртћїърѕ. быхфѓўљшщ я№шьх№ яюърцхђ, ъръ ъртћїъш ш ъюэёђ№ѓъішш схч ъртћїхъ тючт№рљрўђ ёђ№юъш (юёюсхээю трцэю, ъюуфр т §ђшѕ ёђ№юърѕ хёђќ я№юсхыћ):


#!/bin/bash

 

array=("first item" "second item" "third" "item")

 

echo "Number of items in original array: ${#array[*]}"

for ix in ${!array[*]}

do

    printf "   %s\n" "${array[$ix]}"

done

echo

 

arr=(${array[*]})

echo "After unquoted expansion: ${#arr[*]}"

for ix in ${!arr[*]}

do

    printf "   %s\n" "${arr[$ix]}"

done

echo

 

arr=("${array[*]}")

echo "After * quoted expansion: ${#arr[*]}"

for ix in ${!arr[*]}

do

    printf "   %s\n" "${arr[$ix]}"

done

echo

 

arr=("${array[@]}")

echo "After @ quoted expansion: ${#arr[*]}"

for ix in ${!arr[*]}

do

    printf "   %s\n" "${arr[$ix]}"

done
