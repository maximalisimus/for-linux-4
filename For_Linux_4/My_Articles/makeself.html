#bash #makeself #makeselfsh #SFXLinux

 SFX архивы в Linux.
 Да есть 7z и p7zip. Но в данном варианте вы не ограничивпетесь одним единственным архиватором и уровнем сжатия, а также сможете добавить свой собственнный функционал.

 У всего есть недостатки, как и неоспоримые преимущества. В данном случае создание GUI ложится на ваши плечи. Однако, можно выкрутится простым скриптом с использованием Zenity и при создании архива указать этот скрипт в качестве запускаемого по умолчанию при старте архива.

 На заметку. Необязательно устанавливать программу. Достаточно иметь 2 скрипта: makeself и makeself-header. При создании архива их оьязательно надо поместить внутрь этого архива, чтобы иметь возможность полученный итоговый sfx файл запускать на любом ПК. Можете просто скачать последнюю Release версию из Github и запустить её. Прикол в том, что все эти готовые Release сборки по сути и есть самораспаковывающиеся архивы с необходимыми файлами для работы. Распаковываются в текущую директорию. Хотите запускать из любого места - сделайте ln -s ссылку в папку / usr / bin / на сам скрипт makeself.
 Работает даже в GIT-Bash-Windows, что не может не радовать.
